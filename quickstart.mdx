---
title: "Quickstart"
description: "Get going in minutes"
---

1) Sign in
- Open the app → Sign In → enter email/password → verify → Login
- You’ll land on the dashboard with a default org selected

2) Create a workspace
- Click New Workspace → pick services (repo/upload/empty/database/image) → name it
- You’ll be taken to the Workbench

3) Run
- If prompted, add basic start/build commands
- Click Run to start all services; logs and Preview open automatically

4) Edit
- Expand a service → open a file → edit → save

5) Terminals
- Click “+” in the terminal area → choose a service → run commands

That’s it. For a full walkthrough (sessions, merging, tips), see User Journey.
